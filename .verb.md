## Install

```sh
$ docker pull stefanwalther/sense-go
```

## Run

```sh
$ docker run --rm -v ${PWD}:/opt/sense-go stefanwalther/sense-go <PARAMS>
```

To execute the `build` task, use:

```sh
$ docker run --rm -v ${PWD}:/opt/sense-go stefanwalther/sense-go build
```
